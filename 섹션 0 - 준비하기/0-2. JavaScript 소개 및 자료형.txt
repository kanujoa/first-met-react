1. Javascript : 스크립트 언어
   * Script Language : 프로그램이 실행되는 런타임에 코드가 해석되고 실행됨. C나 JAVA에서는 컴파일 과정으로 실행 가능한 형태로 소스 코드
                       가 변환된 후 실행됨.

2. 문법
<자료형>
- JS에서는 변수를 선언할 때가 아닌 변수에 자료를 대입할 때 해당 변수의 자료형이 결정됨.
  --> 동적 타이핑(Dynamic Typing) 
  --> Number type, String type, Boolean type, Null type(값이 정의되긴 하였지만 그 값이 null(값이 없음)), Undefined type
      (변수를 선언만 하고 값을 대입하지 않았을 경우), Array type, Object type 

- JS에서 객체는 다른 객체 지향 프로그래밍 언어에서의 객체와 좀 다름. key와 value로 이루어진 쌍의 집합이 객체이다.
  Python dictionary와 비슷한 개념!
